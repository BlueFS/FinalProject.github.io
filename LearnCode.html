Simple HTML Commands

Tag Name What It Does
<html> The root element; it defines it as a HTML document.
<head> Contains metadata/information for the document.
<title> Defines a title for the document.
<body> Defines the document's body.
<h1> to <h6> Defines HTML headings which are listed from 1 to 6.
<p> Defines a paragraph.
<br> Defines a line breaak.
<b> Defines bold text.
<i> Defines italicized text.
<b> Defines bold text.
<a> Defines an anchor/hyperlink; this can be used for links.
<nav> Defines naviation links.
<table> Defines a table
<img> Defines an image.
<audio> Defines an sound media.
